Installation

Clone repository:

git clone <your-repository-url>
cd <project-folder>

Install dependencies:

pip install -r requirements.txt
Environment Setup

Create .env file:

HUGGINGFACEHUB_API_TOKEN=your_token_here
Run the Project
python main_final.py
